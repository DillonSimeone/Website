Option A: OBS Link

Route your entire OBS Studio scene (and audio) directly into this page.

  • In OBS: Click Start Virtual Camera
  • Use VB-Cable to route OBS Audio to a virtual input.
  • Select "OBS Virtual Camera" & "CABLE Output" here.

Option B: Loopback Audio

Standard computers can't "hear" themselves. You need a Virtual Loopback driver to route your desktop sound into this app.

  • 1. Install: Download VB-Cable or AudioRelay.
  • 2. Configure: Set the tool to capture "Desktop Audio" and output it as a "Virtual Microphone".
  • 3. Select: Choose that new device in the Audio Input list here.

The Diagnostic Trick:

Play a video or music, then cycle through your Audio Input list. When the visualizer bars below start "wiggling," you've found an active loopback! (Common names: CABLE Output, Virtual Mic, Stereo Mix).
