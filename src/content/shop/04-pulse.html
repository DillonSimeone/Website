PULSE: Haptic Ecosystem

Price: $30.00 (Leader) / $60.00 (Follower)

The PULSE ecosystem is a professional-grade hardware and software framework for high-fidelity tactile feedback. This system uses an ESP32-based "Leader" unit to broadcast commands to multiple "Follower" haptic saucers via ESP-NOW, creating a synchronized, low-latency haptic environment.

The Follower units feature a high-durability mechanical design and a refined haptic response, perfect for music visualization, gaming, or accessibility research.
