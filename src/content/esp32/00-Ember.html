Ember: Bytecode VM

A state-of-the-art stack-based bytecode virtual machine and Pratt-compiler for real-time LED pattern generation on the ESP32-C3. Ember features a FreeRTOS-isolated render pipeline, zero heap allocations, 10x faster Lookup-table waveforms, a real-time 8-band FFT audio analyzer, and a live sandboxed playground.
