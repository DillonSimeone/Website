# Pico LED Rainbow Test

## Overview
A simple hardware verification project for the **Raspberry Pi Pico**. It implements a high-refresh-rate rainbow animation using the FastLED library.

## Hardware Setup
*   **Microcontroller:** Raspberry Pi Pico (RP2040)
*   **LED Strip:** WS2812B (Pin 4)

## Key Features
*   **FastLED on RP2040:** Demonstrates compatibility of the FastLED library with the Pico's architecture.
*   **Rainbow Cycle:** A standard shifting rainbow pattern to verify timing and power stability.
