# esp32-2432s028r-espnow — Cost-Down Analysis
2432S028R is a popular "cheap yellow display" board ($8 complete). For prototyping volumes under ~100, keep the integrated board — you can't beat its silicon-plus-display-plus-touch price. For larger volumes, design a custom PCB using ESP32-C3 + ILI9341 2.8" + XPT2046 touch.
